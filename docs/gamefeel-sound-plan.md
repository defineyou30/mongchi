# 게임필·사운드 작업계획

2026-07-07. 상호작용 전수 감사(코드 실측) + 설계 제안 검증 + 사운드 계획. 분석 원본: workflow wto3icty9.

## 0. 핵심 발견 (실측)

1. **앱이 완전 무음이다** — 오디오 라이브러리 0개, 사운드 에셋 0개, 알림조차 sound:null. **햅틱도 0건** (expo-haptics 미설치).
2. **이미 만들어놓고 화면에 안 꺼낸 축하가 3개** — ① 산책 귀가 시 도메인이 celebrate 표정+발견 대사("A new find!")+lastWalkDiscovery를 생성하지만 홈이 표시하지 않음(도달 불가 코드), ② 유대 레벨업은 어떤 화면 피드백도 없음(토스트 0), ③ 30일 편지 개봉이 setState 한 번(연출 0).
3. **죽은 아이템/동선**: Buddy Plush·Rose Cushion은 기본 공놀이/쓰다듬기와 체감 차이 0(소비도 고유 반응도 없음). rest 액션은 도메인 완성+버튼 에셋까지 있는데 dock에 미렌더. 인벤토리는 열람 전용(카드가 Pressable도 아님). 채팅 Mic 버튼은 장식.
4. 토스트 대부분 등장 애니메이션 없음(HomeEventToast만 있음), HUD 게이지는 즉시 리렌더(차오르는 연출 없음), 사이드 레일 버튼 pressed 상태 없음.

## 1. 게임필 로드맵 (검증 통과분, 전부 소품 피벗·죄책감 금지와 무충돌)

### Tier 1 — 이미 만든 것 꺼내기 (JS-only, 즉효)
- **산책 귀가 마중**: Greet & claim 시 펫이 뛰어 들어오는 연출(발자국 Lottie 재활용) + celebrate 표정 + 도메인이 이미 만드는 발견 대사를 말풍선에 표시 + 새 도감이면 발견 카드/완성 시 축하(+20cr이 조용히 지급되는 현재 문제 해결)
- **유대 레벨업 모먼트**: HomeEventToast + celebrate 표정 + 햅틱(추후)
- **HUD 게이지 필 애니메이션**(0.6s) + 델타 플로터("+12"), careFeedback 토스트에 등장/퇴장 애니메이션(HomeEventToast 패턴 이식)
- **편지 개봉 연출**: 봉투 → 펼침 전환(LayoutAnimation/Reanimated)
- 레일 버튼 pressed 상태, 표정 팩 해금 리빌(새 표정 3종 살짝 쇼케이스)

### Tier 2 — 케어 순간 컨텍스트 연출 (승인된 방향)
- **밥그릇**: 등장(scale+fade) → 사료 채워짐 → 펫 eating 상태(표정 팩 파이프라인으로 상태 1종 추가) → 빈 그릇 → 퇴장
- **물그릇**: 동일 패턴 + drinking 상태 (에셋 drinkWaterBowl 이미 존재)
- **공 던지기**: 공 포물선+바운스, 펫이 쫓아가서 물고 돌아옴(좌우 반전+translateX) — 확장: 실패 없는 3회 왕복 캐치(탭하면 다시 던짐, 판정·점수 없음)
- **쓰다듬기 드래그**: 펫 위 Pan 제스처 → 하트 파티클 + content 상태. 기존 탭=affection 유지, 드래그는 리치 버전 (백로그 §10과 일치)

### Tier 3 — 세계의 자율성
- **시간대 연출**: 22~6시 수면(sleep 표정 이미 무료 3종에 있음!) + 남색 오버레이 + zzz, 밤 케어는 살짝 눈 뜨고 받아주고 다시 잠(벌 없음). 아침 첫 진입 기지개 (백로그 sleeping_night 항목과 정렬)
- **자율 행동 확장**: 기존 30초 idle 펄스(curious↔happy)를 확장 — 하품/몸 털기/뒹굴기/화면 안 자리 이동(2프레임 상태 + translateX)
- **나비 방문객**: 낮 ~20% 확률로 나비가 가로질러 날고, 탭하면 펫이 시선 추적 + 코끝에 앉는 희귀 연출 + 전용 대사 → 기억 스파인에 기록 가능
- **숨은 간식 찾기**: 하루 1회 배경에 스파클 → 탭하면 펫이 달려가 dig → 간식 1개 + 대사. 페널티·타이머 없음

### Tier 4 — 아이템 개별성
- **장난감별 고유 반응**: Plush 선택 시 물고 흔드는 연출 + 전용 대사 풀 — 신상 장난감 = 연출+대사 템플릿 (사용 횟수는 careStats에, 기억 스파인 아님)
- **간식 취향**: 이미 있는 favoriteTreat 파이프라인(개별성 카드·편지·채팅에 연결됨)에 **반응 차이만 배선** — favorite 간식이면 excited 점프+특별 대사
- **인벤토리 Give now**: 카드 탭 → 홈으로 이동+트레이 자동 오픈
- **rest 부활**: 쿠션 낮잠 — 쿠션 등장 + nap 상태 + 화면 살짝 어두워짐 + 에너지 회복 (도메인·에셋 완성 상태)

## 2. 사운드 작업계획

**아키텍처**: expo-audio (expo-av는 deprecated) + expo-haptics — 둘 다 네이티브 모듈이라 **dev client 재빌드 1회 필요 → 위젯 등 다른 네이티브 작업과 반드시 같은 빌드로 묶을 것**. 사운드 매니저 구조: `apps/mobile/src/shared/audio/` (soundManager/sfxPlayer/bgmPlayer/ambiencePlayer/audioAssets 매니페스트/useAudioSettings).

**에셋 23종** (P0 12종): 공통 탭 블립, 케어 5종 SFX(사료 또르르/공 바운스/목줄 딸랑/하트 차임/물 따름), 간식, 토스트 차임, 징글 3종(레벨업 아르페지오/도감 발견 반짝/편지 개봉 — 가장 공들일 것), 리빌 원샷. P1: 타자기 블립(2-3글자당 1회 레이트리밋), 산책 귀가, 구매(카지노 느낌 회피), BGM 데이/나이트(뮤직박스·칼림바 로파이 BPM 70대), 새소리/빗소리 앰비언스(날씨 연동). P2: 바람/귀뚜라미, 테마별 앰비언스 1레이어(풀 BGM 변주 대신 — 작업량 1/5), 강아지 시그니처.

**⚠️ 강아지 소리 원칙**: 사실적 짖음 금지 — 아바타가 "유저의 실제 강아지"라서 남의 개 목소리는 몰입 파괴. 비성문(목걸이 딸랑, 발소리, 천 부스럭) 폴리만.

**소싱**: Kenney(CC0)/freesound(CC0) + BeepBox 직접 시퀀싱(징글 패밀리 통일), BGM은 itch.io 로열티프리 팩 소액 구매($10-20) 권장. Suno 등 AI 음악은 저작권 지위 불안정으로 폴백만.

**단계**: Phase 0 재빌드(0.5일, 위젯과 합침) → Phase 1 핵심 SFX+햅틱 배선(~3일: P0 12종, 케어 성공 콜백에 1줄씩, 햅틱 페어링, Sounds 토글) → Phase 2 BGM+앰비언스(~2.5일: 시간대 크로스페이드, 날씨 구독, 덕킹, Music 토글) → Phase 3 날씨/테마 매트릭스+믹스 패스(~2-3일).

**설정/접근성**: Settings "Little reminders" 아래 "Sound & feel" 섹션 — Sounds / Music & ambience 2토글(3단 분리는 과함), 기본 ON, AsyncStorage 로컬 프리퍼런스. **iOS 무음 스위치 = 마스터 뮤트**(ambient 카테고리), **mixWithOthers 필수** — 유저가 듣던 팟캐스트/음악을 절대 끊지 않기 (QA 체크리스트: "Spotify 재생 중 앱 실행 → 음악 안 끊김").
