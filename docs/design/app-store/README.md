# Mongchi App Store Screenshots

This folder keeps the approved App Store visual set and the structural reference used to build it.

## Final exports

The release-ready English set is in `outputs/v2/`:

1. `01-hero-rich.png`
2. `02-home-care-mockup.png`
3. `03-one-photo-tiny-self.png`
4. `04-talk-about-your-day.png`
5. `05-every-day-memory.png`
6. `06-meet-every-side.png`

All six exports are 1320 x 2868 px, opaque, and sRGB. `contact-sheet-six.png` is the review sheet, and `final-qa.md` records the final visual checks and hashes.

## Reference and working files

- `references/moshi-kids-structure/` preserves the cropped reference strip and its four readable panels.
- `work/` and `outputs/v1/` remain local because they contain masks, intermediate crops, previews, and superseded compositions.
- Local working files must not be used as App Store upload artifacts.

Localized screenshot sets should be exported to a locale-specific folder such as `outputs/ko-KR/`, while preserving the six-slide order and visual composition.
