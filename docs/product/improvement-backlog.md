## 1. 캐릭터 표현력 (같은 문구 반복 문제)

- [x] **문구 풀 절대량 부족** — 반응 규칙 42개(ko 27 / en 17), 총 문구 약 100줄. 밥주기 등 각 행동당 문구 세트가 사실상 1개라 쿨다운마다 같은 말 반복. → 규칙 카탈로그 대량 확장 (`expandedReactionRules.ts`).
- [x] **상태 문구가 20분 단위 해시 고정** — `petStatusEngine.ts`의 `seedTimeKey`가 20분 창 안에서 항상 같은 문장 선택. → 시딩 창 축소 + 케어 상태 엔트로피 추가 + 문구 풀 확장.
- [x] **퍼센테이지 무시** — 포만감 0과 45가 같은 대사. 스탯 구간(critical/low/okay/great) 개념 없음. → `getCareStatBand()` 도입, 구간별 대사 분리 (배고픔 "살짝" vs "매우").
- [x] **성격/말투가 대사에 미반영** — `talkingStylesAny`, `species` 조건이 엔진에 있으나 사용하는 규칙 0개. 어떤 성격을 골라도 같은 펫. → 성격 6종 × 시간대, 말투 4종 변형, 종별(개/고양이) 규칙 추가.
- [x] **미구현 카테고리** — `play_done`, `treat_special`, `walk_return_rare` 카테고리는 정의만 있고 규칙 없음. → 규칙 추가.
- [ ] **`memoryNote` 활용처 없음** — 필드만 존재. 추후 대사 플레이스홀더(`{memoryNote}`) 및 채팅 컨텍스트에 주입.
- [ ] **계절 인식 없음** — `seasonal` 스프라이트/아이템이 날짜 기반으로 활성화되지 않음. 월 기반 시즌 판정 + 시즌 한정 대사/배경 팩 필요.

## 2. 상태가 몸으로 드러나지 않는 문제

- [x] **방치해도 웃는 펫** — 스탯이 바닥이어도 표정 변화 없음. sad/sick/messy 스프라이트 상태가 없음. → `generatedAssetStates`에 3종 추가 + 폴백 체인(sad→hungry, messy→walk_return 등) + 케어 구간 기반 표정 자동 결정(`deriveAmbientPetAssetState`).
- [x] **생성 파이프라인에 신규 상태 누락** — 사진 → 캐릭터 생성 시 뽑아야 할 이미지 목록에 sad/sick/messy 추가 (프롬프트 정의 포함).
- [ ] **신규 상태 실제 이미지 에셋** — miso/luna 폴백 PNG 3종(sad/sick/messy)은 이미지 생성 필요 (`npm run generate:mobile-assets` 계열 파이프라인으로 생산 후 `generatedPetAssets.tsx`에 등록).
- [ ] **날씨 파티클 없음** — `WeatherSceneLayer`가 패럴랙스만 렌더. 비/눈 파티클(Lottie) 추가하면 체감 큼.

## 11. 생성 비용 최적화

- [x] **기본 품질 low 전환** — `openAiImageProvider` 기본 quality medium→low (출력 단가 약 1/4). `TINY_PET_WORKER_IMAGE_QUALITY` 환경변수로 auto/high/medium/low 오버라이드 가능.
- [x] **스프라이트 시트 모드** — `statesPerSheet` 옵션(4=2×2@1024², 6=3×2@1536×1024, 셀 최소 512px). 시트 1회 호출 후 sharp로 셀 슬라이싱. 프로덕션 런타임 기본값 4 (호출 16→4회). `TINY_PET_WORKER_IMAGE_STATES_PER_SHEET`(1/4/6)로 조정. 예상 유저당 이미지 원가 ~$0.70 → **~$0.08**.
- [ ] **실사진 검증** — 시트 모드는 모델이 그리드 배치를 지키는지에 품질이 좌우됨. `aiGenerationProofHarness`로 실제 사진 몇 장을 1장/4장/6장 × low/medium 매트릭스로 뽑아 비교 후 기본값 확정 필요. 셀 경계 침범 시 품질 게이트에서 걸러지지만, 슬라이싱 blob 검증(투명 배경 연결요소 분석) 추가하면 더 견고해짐.
- [~] **재생성 횟수 제한 정책** — 크레딧 서버 원장(`credit_wallets`/`credit_ledger`, 마이그레이션 0004) + RPC 4종 + 표정팩 서버 선차감 게이팅 랜드(커밋 1114ff9·f881743·c5d405c). 무료 60개 소진 후 크레딧 소비처는 있으나 **버는 곳·사는 곳(크레딧 IAP SKU·데일리 파우셋)이 없어 경제가 데드엔드** — 출시 전 매출 급소로 남음.
