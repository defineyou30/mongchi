# 한 달 리텐션 갭 분석 — "내 반려동물이 사는 작은 세계"

2026-07-04. 25개 에이전트 다각 분석 (콘텐츠 실측 → 5렌즈: 여정 시뮬레이션/장르 벤치마크/정서 유대/앱 밖 존재감/BM-리텐션 → 코드 대조 검증). 수치는 전부 코드 실측.

## 핵심 결론

**현재 코드 기준 D30 리텐션을 지탱하는 시스템은 없다.** 근본 원인은 콘텐츠 양이 아니라 구조 2가지:

1. **시간이 흘러도 아무것도 생기지 않는다** — 모든 콘텐츠가 출하 시점 고정 자산. 부재 중 생성되는 것 0, 강아지가 먼저 시작하는 사건 0, 시즌/주간 갱신 경로 0. "Stories piled up while you were away" 문구 뒤에 실제 쌓인 콘텐츠가 없음.
2. **경제가 일방향이다** — 무료 크레딧 평생 총량 60cr(시작 25 + L5 5 + L10 10 + 도감 20), 반복 수입 0. 심지어 지불 의사가 생겨도 크레딧 팩 IAP SKU가 코드에 없음.

## 검증된 갭 (18건 중 15건 코드 대조 검증, 3건 세션 한도로 미검증)

### A. 출시 전 필수 (now) — 사실상 버그

1. **알림 파이프라인 전 구간 무기능** [critical/high]: 퍼미션 요청 함수가 설계·테스트까지 완비됐지만 **호출하는 UI가 0건**(데드 코드) → 프로덕션 알림 0건. 스케줄러도 "마지막 세션+5분"만 예약해 이탈 유저에게 구조적으로 침묵. launch-plan에는 ✅완료로 오표기.
2. **날씨가 좌표 해시로 영구 고정** [critical/high]: `seed = round(lat*10)*31 + round(lng*10)*17` — 시간 성분이 없어 이사하지 않는 한 영원히 같은 날씨. 도감 완성(최대 파우셋 20cr)이 대부분 유저에게 도달 불가(기본 설정 유저 4/9 영구 정지)이고, 날씨 대사·배경 변화 전체가 정적.
3. **테마 BM 총체 결함** [major]: 같은 화면에 fairy garden 18cr 구매 카드와 4종 전부 'Free' 즉시 적용 목록이 공존. 구매해도 소유 기록이 없어 재적용 시 재과금 구조. 날씨별 배경 키 18개가 전부 같은 이미지 1장.
4. **표정 수렴 + 유료 퍼널 부재** [major/high]: 파이프라인은 16종 생성 가능한데 무료 3종에 반응이 수렴(실측: 산책귀가/간식/축하/배고픔이 사실상 idle 1종). 표정 팩 판매 코드 0줄 — 소품 제거 후 표정이 유일한 시각 콘텐츠인데 1일차에 플랫라인.
5. **스트릭 하드 리셋** [major/high]: 하루 공백 시 무통보 0 리셋, 프리즈/그레이스/위험 알림 전무(Duolingo가 검증한 최대 이탈 트리거). 보상도 3/7일 무한 반복이라 7일차에 패턴 소진.
6. **크레딧 경제 양방향 동결** [critical]: 반복 수입 0 + 크레딧 팩 SKU 0. launch-plan의 "재생성 ~2주 1회"는 산술적으로 불가능.
7. **역방향 돌봄 부재** [major]: 강아지가 유저의 안부를 묻지 않음(Finch의 D30 엔진). 유일한 'your day' 질문이 유료 채팅 안에 있음.
8. **공유·저장 표면 부재** [major]: 리빌 순간(공유 의욕 최고점)에 저장/공유 버튼 0. 퍼미션 문구만 선언됨.
9. **개인화 데이터 사장** [major]: 온보딩에서 받은 favoriteThing이 무료 표면 대사에 0줄(talkingStyle 룰은 8개 존재하나 커버리지 최소).

### B. 출시 후 첫 업데이트 (post-launch)

10. **기억 부재** [critical/high]: 관계 기억 축적·유대 깊이별 대사 분화가 타입 수준에서 불가능(ReactionConditions에 bondLevel/daysTogether/과거 이벤트 조건 필드 없음). MemoryNote 도메인은 데드 코드.

12. **마일스톤 휘발** [major]: 첫 레어 발견·도감 완성·7일 스트릭이 어디에도 기록 안 됨. memories 배열+친구 페이지 카드로 서버 없이 가능.
13. **유대 레벨업이 지속 변화 0** [major/high]: 소모품+축하 한 줄뿐. bondLevel별 대사 팩 해금이 최소판(며칠 규모).
14. **3-4주차 콘텐츠 절벽** [critical→medium 조정]: L10(29일)에서 모든 트랙 종료. 성실 코호트의 LTV 타격.
15. **위젯/Live Activity 부재** [major/high]: 장르 최강 터치포인트(Widgetable/Finch 검증). 서버 불요, 단 bare 워크플로라 네이티브 타깃 추가 필요(실현성 medium).

## BM-리텐션 상호작용 (경제 렌즈 결론)

- 유료 콘텐츠 전량 26cr ≈ 시작 지급 25cr → 첫 주에 상점 소진
- 현재 구매는 리텐션을 **끝내는** 구조: 테마는 1탭 소비, 간식 구매는 마지막 트랙(유대)을 가속, 소유 기록 없음
- 1인 개발에서 지속 가능한 콘텐츠 축은 "월 N종 테마"가 아니라 **표정 팩(매 순간 노출) + 시즌 도감 교체(기존 시스템 재활용)**

## 최소 수술 로드맵 (제안)

- **W-알림/날씨/스트릭**: 퍼미션 배선(수 시간) + 날씨 시드에 시간 성분 + 스트릭 그레이스 1일 + 위험 알림
- **W-표정**: 무료 3종 반응 배선 강화 + 표정 팩 수직 슬라이스 1개(생성→구매→해금 E2E)
- **W-테마/경제**: 소유 기록 + Themes 탭 단일화 + 크레딧 팩 SKU + 반복 파우셋 1개(예: 데일리 체크인)
- **W-관계**: 데일리 체크인(역방향 돌봄) + 메모리 축적(이벤트→기록→다음날 회귀 언급) + bondLevel 대사 조건
- **W-외부**: 리빌 공유 카드 → Small 위젯
