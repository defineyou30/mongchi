## North Star

```text
My pet has a tiny world in my phone, and it knows me.
```

## Core Product Decisions

- Product type: pet-life game plus healing companion.
- Main subject: user's real dog/cat photo transformed into a tiny digital avatar.
- Main world: cozy glass-dome terrarium.
- Platform target: iOS and Android.
- Client candidate: React Native with Expo.
- Free speech: unlimited authored/local state-based reactions.
- Premium speech: extended AI chat.
- Items: decoration first, special treats later for cute behaviors.
- Walk: idle send-and-return reward first, deeper adventures later.
