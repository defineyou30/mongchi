# Mobile Fonts

## Locale mapping

- Latin locales (`en-US`, `de-DE`, `fr-FR`, `pt-BR`, `es-MX`): Pixelify Sans display with Baloo 2 body.
- Korean (`ko-KR`): Fusion Pixel Font 10px Proportional Korean.
- Japanese (`ja-JP`): Fusion Pixel Font 10px Proportional Japanese.
- Traditional Chinese (`zh-TW`): Fusion Pixel Font 10px Proportional Traditional Chinese.

The Fusion Pixel files are pinned to release `2026.07.01` from
`https://github.com/TakWolf/fusion-pixel-font`. The font and its bundled source
fonts are distributed under SIL Open Font License 1.1; license texts are in
`licenses/FusionPixelFont-*.txt`.

All non-ASCII characters currently used by the Korean, Japanese, and
Traditional Chinese translation resources are present in their mapped font.
The Korean build also contains all 11,172 modern Hangul syllables.
